# Post cover images

Drop press/insight cover images here, then reference them in the portal's
**Cover image** field as either:

- a full URL — `https://…`, or
- a site path — `assets/post-covers/your-file.jpg`

The Press & Insights cards show the cover as a banner above the title. If a
post has no cover image, the card falls back to a per-type color scheme
(see `.post-cover--*` rules in `styles.css`).

Recommended: landscape JPG ~1000px wide, under ~250 KB.
