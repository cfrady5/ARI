# About page — team photo wall

These images feed the 3-column vertical scrolling photo wall above the
Leadership section on the About page.

## How to add your photos

Replace the numbered placeholders with your own photos, keeping the **exact
filenames** (`01.jpg` through `27.jpg`) — no code changes needed. Just upload
a file with the same name to overwrite it.

- **Capacity:** up to **27 photos** (`01.jpg` … `27.jpg`).
- **Columns:** `01–09` = left column, `10–18` = middle column,
  `19–27` = right column. The middle column scrolls the opposite direction.
- **Orientation:** portrait works best — tiles are cropped to a **4:5** ratio
  with the subject centered (`object-fit: cover`).
- **Recommended size:** ~800 × 1000 px or larger; `.jpg`.

You don't have to fill all 27 — any slot you leave as the placeholder will
simply show the numbered tile. To use fewer photos cleanly, tell me how many
and I'll trim the wall to match.
